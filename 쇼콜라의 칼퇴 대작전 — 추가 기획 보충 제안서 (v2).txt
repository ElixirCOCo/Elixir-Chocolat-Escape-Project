🎮 쇼콜라의 칼퇴 대작전 — 시스템 설계 제안서
1. 프로젝트 방향
본 프로젝트는 단순한 탈출 게임이 아니라 **“캐릭터 중심 탈출 TPS + 반복 플레이 구조”**를 목표로 한다.
초기 목표는 수업용 프로토타입 완성이며, 이후 점진적으로 확장하여 출시를 지향한다.
2. 핵심 문제 정의
현재 고민

스텔스 + 라우드 구조는 이미 검증된 구조지만
반복 플레이 시 1회성 경험으로 끝날 가능성 존재 즉, “한두 번 재미있고 끝나는 게임”이 될 위험
기본 게임 루프
침투 → 탐색 → 아이템 획득 → 탈출 판단 → 탈출 or 실패
핵심 감정:

긴장 (타이머)
선택 (탈출 vs 추가 탐색)
압박 (적, 환경)
메타 구조 (반복 플레이 유도)
💰 기본 구조

탈출 성공 시 돈 획득
해당 자원으로 장비 / 무기 / 요소 구매
다음 플레이에 반영 👉 구조 자체는 익스트랙션 계열과 유사
차별화 전략 (중요)
❌ 피해야 할 방향

단순 돈 파밍 → 장비 강화
성능 기반 전용무기
컷신/일러스트만 보상
👉 결과: 메타 고정 + 빠른 이탈
✅ 채택 방향
5-1. 캐릭터 중심 설계

캐릭터 = 플레이 스타일
예:

스텔스 특화
전투 특화
해킹 특화
👉 “장비 선택”이 아니라 캐릭터 선택
5-2. 전용무기 설계 원칙
성능 차이 금지 / 감각 차이 허용

DPS 동일
반동 / 연사 / 사운드 / 연출 차이
👉 밸런스 유지 + 체감 다양성 확보
5-3. 씹덕 요소 활용 방식
❌ 잘못된 방식

단순 컷신 / 배경화면 해금
✅ 올바른 방식
“플레이 중 체감되는 요소”

전용 애니메이션
피격 반응
사격 이펙트
탈출 연출 변화
👉 캐릭터를 “보는” 게 아니라 “조작하는 느낌”
5-4. 반복 플레이 유도 시스템
🎯 퍼포먼스 기반 해금

노킬 클리어 → 스텔스 보상
라우드 클리어 → 전투 보상
빠른 탈출 → 속도형 보상
👉 다양한 플레이 스타일 강제 유도
5-5. 감정 기반 보상 시스템
밸런스 영향 없이 몰입 증가

캐릭터 반응 변화
탈출 시 전용 연출
실패 시 코믹 리액션 👉 “한 판 더”를 유도하는 핵심 요소
차별화 핵심 정의
기존 게임

돈을 벌기 위해 싸움
효율 중심 플레이
본 프로젝트
“퇴사하기 위해 싸운다”

목적: 탈출
감정: 불안, 선택, 압박 👉 장르적 차별화 핵심
최종 설계 원칙

밸런스를 건드리지 말 것
감각(손맛, 연출)을 건드릴 것
캐릭터 = 플레이 경험으로 연결할 것
보상은 “수치”가 아니라 “체감”으로 설계할 것
결론
본 프로젝트는 단순한 탈출 게임이 아니라
“캐릭터와 함께 탈출하는 반복형 TPS”
를 목표로 한다.

기본 루프의 재미 확보

반복 동기 설계
========================================================
📄 쇼콜라의 칼퇴 대작전 — 추가 기획 보충 제안서 (v2)

기존 제안서 이후 대화 기반 확장
목적: 반복성, 동기부여, 억까 방지, 시스템 완성도 보강
대상: 개발 AI (Claude / GPT 등)

🎯 1. 핵심 문제 재정의

기존 문제:

탈출 루프가 단순 (사원증 + 키카드)
반복 플레이 동기 부족
페이데이2 유사 구조 위험
랜덤 디버프 → 억까 → 강종 위험

추가로 드러난 핵심 이슈:

1. 반복 플레이 이유 부족
2. 실패/디버프가 “부당하게” 느껴질 가능성
3. 장기 플레이 목표(메타 루프) 약함
4. 게임 템포 단조로움
🔥 2. 핵심 해결 구조
🧠 A. “야근 스택 시스템” (Run Persistence Core)

반복 플레이의 이유를 만드는 핵심 시스템

구조:
탈출 성공 시 → 야근 스택 +1

스택 증가 효과:
- 난이도 상승
- 특수 이벤트 활성화
- 보스 해금 조건 진행
핵심 포인트:
단순 반복이 아니라 “누적 긴장”
유저는 “더 높은 스택을 찍고 싶다”는 욕망 생성
👑 B. 보스 해금 구조 (장기 목표)
스택 100% 도달 시 → 보스 스테이지 입장 가능 (선택)
특징:
강제 아님 (유저 선택)
리스크 vs 보상 구조 유지

👉 효과:

“왜 계속 탈출해야 하는가?”에 대한 명확한 답 제공
⚡ C. 스택 가속 시스템 (하이리스크 선택)
유저가 난이도 가속 선택 가능
→ 더 빠르게 스택 증가
→ 대신 난이도 급상승

예시:

적 스폰 속도 증가
타이머 감소
엘리트 적 등장

👉 핵심:
“너가 선택한 고통이다” 구조

🎲 D. 확률 이벤트 시스템 (리워크)

기존:
❌ 랜덤 디버프 (억까 발생)

개선:
✅ 선택 기반 리스크 시스템

구조 변경:
스택 상승 시 → 선택지 제공

[야근 옵션 선택]
1. 보상 증가 / 대신 적 강화
2. 아이템 확률 증가 / 대신 타이머 감소
3. 시야 감소 / 대신 공격력 증가

👉 핵심:

랜덤 제거
선택 기반 설계
🛡️ E. 억까 방지 시스템 (Anti-Frustration Layer)
필수 보호 규칙:
체력 30% 이하 → 추가 디버프 금지
연속 피격 → 일정 시간 보호
타이머 임계치 → 강한 디버프 제한
RNG 제어:
강한 디버프 연속 등장 금지
상황 기반 확률 조정

👉 목적:
“죽어도 납득하게 만든다”

💊 F. 디버프 해소 수단 (Recovery Loop)

디버프는 반드시 풀 수 있어야 한다.

예시:
특정 아이템: 디버프 제거
적 처치 시: 디버프 완화
생존 시간 유지: 자연 해제

👉 핵심:
절망 상태 방지

🧩 G. 디버프 설계 방향 (중요)

❌ 금지:

단순 능력치 감소 (속도↓, 데미지↓)

✅ 권장:

플레이 스타일 변화
예시:
반동 증가 → 대신 크리티컬 증가
시야 감소 → 대신 적 위치 표시
재장전 느림 → 대신 데미지 증가

👉 결과:

불쾌감 ↓
재미 ↑
💰 H. 메타 보상 시스템 (반복 동기)
구조:
탈출 → 재화 획득 → 다음 런에서 사용
사용처:
무기 해금
장비 구매
퍽 시스템
🎭 I. 감성 보상 (차별화 핵심)

밸런스를 건드리지 않는 보상

요소:
캐릭터 의상
컷신 해금
탈출 연출 변화
모션
무기 스킨

👉 핵심:
“성능이 아니라 욕망을 건드린다”

🧍 J. 캐릭터 확장 시스템
VRC 아바타 다양화
캐릭터별 개성
호감도 시스템 (단, 밸런스 영향 없음)

👉 역할:

씹덕 감성 타겟팅
장기 플레이 유도
🎮 K. 모드 구조 (확장성)
기본:
스텔스 중심 탈출
확장:
라우드 모드 (전투 중심)
익스트랙션 모드 (타르코프식)

👉 핵심:
초기엔 단순 → 이후 확장

🧠 최종 설계 철학
1. 반복할 이유를 만들어라 (스택 + 보스)
2. 선택하게 만들어라 (리스크 선택)
3. 억까를 제거하라 (보호 시스템)
4. 실패를 납득 가능하게 만들어라
5. 보상은 성능 + 감성 둘 다 잡아라
🚨 개발 시 절대 금지
❌ 완전 랜덤 디버프
❌ 회복 수단 없는 패널티
❌ 반복 의미 없는 루프
❌ 밸런스를 깨는 캐릭터 보상
🎯 결론

이 시스템은 단순 탈출 게임이 아니라:

“선택 → 후회 → 재도전”을 반복시키는 구조
